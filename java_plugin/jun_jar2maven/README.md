# jarToMaven

#### 介绍
jar包转成maven项目的pom.xml格式
将网上的代码整理一番，供网友直接下载运行


#### 软件架构
软件架构说明


#### 安装教程

1.  运行pom.xml把需要的jar包下载下来。
2.  运行里面唯一的java文件。
3.  控制台输出结果，红色内容为未找到该jar所对应的格式，需要自己上网找替代版本或pom.xml配置本地jar包

#### 软件运行环境说明

1.  Eclipse
2.  jdk1.7
3.  联网
