DBDocTool
=========

数据库文档自动生成工具

http://xiruibing.github.io/DBDocTool/

该项目将持续更新，计划提供各种模版，实现自定义模版

支持单表或表集合生成

方便程序员书写数据库文档

把数据库表结构信息转化为数据库文档

支持命令行操作
支持以下数据库：
Mysql、SQL Server、PostgreSQL、DB2、oracle
